# 🚀 docker-cicd

A minimal Dockerized Flask application designed for CI/CD pipeline demos and experimentation.

---

## 📖 Introduction

**docker-cicd** is a simple Python Flask web app containerized with Docker. This repository serves as a practical template for learning, testing, and implementing Continuous Integration and Continuous Deployment (CI/CD) pipelines using Docker.

---

## ✨ Features

- 🐳 **Dockerized**: Easily build and run in isolated environments.
- ⚡ **Minimal Flask App**: Clean, lightweight, and easy to extend.
- 🔁 **CI/CD Ready**: Perfect base for automating deployments.
- 🚀 **Quick Start**: No unnecessary dependencies or bloat.

---

## 🛠️ Installation

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/) installed on your machine.

### Steps

1. **Clone the repository:**
    ```bash
    git clone https://github.com/sakthivel-26/docker-cicd.git
    cd docker-cicd
    ```

2. **Build the Docker image:**
    ```bash
    docker build -t docker-cicd .
    ```

3. **Run the container:**
    ```bash
    docker run -p 5000:5000 docker-cicd
    ```

---

## 🚦 Usage

Once the container is running, open your browser and navigate to:

```
http://localhost:5000
```

You should see:

```
Hello from Docker CI/CD Pipeline 🚀
```
